 </br>
</br>

### Installation


 Install Flask packages using pip:
 ```sh
pip install -r requirements.txt
```

Install node packages in Client directory: 

 ```sh
npm install
```
Start server in Server directory:
 ```sh
py app.py
```

Start client in Client directory:
 ```sh
npm start
```
